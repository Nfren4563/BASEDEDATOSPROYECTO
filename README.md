# BASEDEDATOSPROYECTO

El proyecto de base de datos constaba de una aplicacion de inventario que gestionaba una base de datos que el profesor nos dio, en la aplicacion el cliente debe de ser capaz de iniciar sesion con diversos usuarios y que cada uno de estos tenga un rol/permisos distintos en la aplicacion, la aplicacion deberia permitir al usuario agregar/eliminar productos y almacenes y manejar la base de datos de una manera sencilla y eficaz, asi tambien no tener errores o fallas con los datos al momento de hacer modificaciones, la aplicacion tambien guarda los datos de los usuarios que modifican los productos, la hora y el dia en el que se realiza el cambio, todo esto almacenado tambien en la base de datos InventarioBD.

Al agregar o modificar los productos un formulario se abre con los datos a ingresar para agregar/modificar un producto, en dado caso que el usuario no tenga permisos para hacer dicha accion estos formularios no seran accesibles.

Este proyecto nos permite a los alumnos aprender mas acerca del manejo de base de datos y entender como funcionan los programas conectados a base de datos junto con los lenguaje de base de datos.
